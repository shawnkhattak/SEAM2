# UI Layout Direction

## Visual Style

Use a clean professional SaaS dashboard style.

Avoid:

- Overly dark hacker aesthetic.
- Too many colors.
- Giant unreadable tables.
- Complicated frontend state.

Use:

- White and neutral cards.
- Clean sidebar.
- Simple charts.
- Good spacing.
- Clear empty states.
- Clear error states.

## Empty State Examples

If no risk flags:

```text
No risk flags found from current data sources.
```

If no ownership data:

```text
Ownership/manager data is not available from the current source for this vessel.
```

If API is not configured:

```text
Oceans-X API key is not configured. Add OCEANS_X_API_KEY to your .env file.
```
